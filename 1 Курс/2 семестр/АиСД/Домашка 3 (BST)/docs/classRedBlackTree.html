Detailed Description

template<typename T, typename T0 = std::enable_if_t<is_comparable<T>::value>>
class RedBlackTree< T, T0 >

Класс красно-чёрного дерева.

Template Parameters
TТип данных, хранящихся в узлах дерева.
T0Вспомогательный параметр для SFINAE (не используется напрямую).

Если операторы сравнения отсутствуют, тип T не сможет быть использован в RedBlackTree.

Перечисление цветов узлов красно-чёрного дерева.

Рекурсивный вывод дерева в скобочной нотации.

Parameters
outПоток вывода.
nodeТекущий узел для вывода.

Формат вывода: (цвет:ключ левое_поддерево правое_поддерево) Пример: (B:10 (R:5 (B:3) (B:7)) (B:15))

Особенности:

  • NIL-узлы не выводятся
  • Цвет узла обозначается префиксом (R: для красного, B: для чёрного)
  • Поддеревья разделяются пробелами

Выполняет левый поворот вокруг указанного узла.

Parameters
pivotУзел, вокруг которого выполняется поворот (ось поворота)

Графическое представление операции: 

*       До поворота:           После поворота:
*
*         pivot                   child
*        /     \                 /     \
*       A     child     -->     pivot     C
*            /     \         /     \
*           B       C       A       B
*
Note
Особенности операции:
  1. Узел child становится новым корнем поддерева
  2. Поддерево B становится левым потомком pivot
  3. Сохраняется порядок элементов: A < pivot < B < child < C
Warning
Ограничения:
  • Запрещено вызывать для NIL-узлов
  • Узел должен иметь правого потомка (child != NIL)
See also
rotateRight() для симметричной операции
