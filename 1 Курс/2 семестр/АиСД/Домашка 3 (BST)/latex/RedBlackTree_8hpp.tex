\doxysubsection{Detailed Description}
Реализация красно-\/чёрного дерева Красно-\/чёрное дерево -\/ это бинарное дерево поиска, где у каждого узла есть атрибут цвета, при этом\+: 

\begin{DoxyAuthor}{Author}
JDH-\/\+LR-\/994
\end{DoxyAuthor}

\begin{DoxyEnumerate}
\item Узел должен быть либо красным, либо чёрным
\item Корень -\/ как правило чёрный
\item Все фиктивные листья (NIL -\/ узлы) -\/ чёрные
\item Оба потомка каждого красного узла -\/ чёрные
\item Любой простой путь от узла-\/предка до листового узла потомка содержит одинаковое число чёрных узлов
\end{DoxyEnumerate}

ВАЖНО! Как и любое другое бинарное дерево поиска, КЧД может работать только с типами, которые можно сравнивать (указано вторым шаблонным параметром)

\begin{DoxyNote}{Note}
Все операции (вставка, поиск) работают за O(log n) в худшем случае. 
\end{DoxyNote}
