\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename T0 = std\+::enable\+\_\+if\+\_\+t$<$is\+\_\+comparable$<$\+T$>$\+::value$>$$>$\newline
class Red\+Black\+Tree$<$ T, T0 $>$\+::\+Iterator}
Итератор для обхода красно-\/чёрного дерева в порядке in-\/order (симметричный порядок). 

Итератор реализует forward iterator, позволяющий последовательно обходить узлы дерева от наименьшего к наибольшему ключу. Обход выполняется нерекурсивно с использованием стека, что гарантирует корректную работу даже для очень больших деревьев.

Особенности реализации\+:
\begin{DoxyItemize}
\item Использует Stack\+Array$<$\+Node\+Ptr$>$ для хранения узлов при обходе
\item Автоматически вычисляет необходимый размер стека на основе количества узлов
\item Поддерживает стандартные операции итератора\+: \texorpdfstring{$\ast$}{*}, -\/\texorpdfstring{$>$}{>}, ++ (префиксный и постфиксный)
\item Корректно обрабатывает end-\/итератор (пустое дерево или завершение обхода)
\end{DoxyItemize}

Гарантии\+:
\begin{DoxyItemize}
\item Время доступа к следующему элементу\+: O(1) амортизированное
\item Потребление памяти\+: O(h), где h -\/ высота дерева
\item Инвалидация итератора при модификации дерева
\end{DoxyItemize}
